    Types of programming languages:


    In the field of computer science, programming languages are used by developers to build a bridge between the computers understanding what developers are trying to achieve. Now in the early days of programming, low level languages such as assembly existed which just allowed computers to perform certain functions using 0’s and 1’s. Eventually as technology developed rapidly, high level languages started to emerge such as Java, C, python and many more. These high-level languages allowed developers to be able to give instructions to computers much easier because high level languages were more understandable by humans.


    Types of Programming paradigms:

    • One of the first paradigm developers used were:
    • Include a user feature that allows users to click on the marker and have a small popup tooltip show


    This type of programming is very simple, however complex programs using this paradigm would cause the whole application to break because when you make a change to one function several other functions break (Spaghetti code). Meaning there are too much interdependency between all the functions created that it becomes inflexible and problematic to fix bugs and change code.

    Eventually the Object-Oriented Programming (another programming paradigm), was created to solve this problem. Object-Oriented Programming is the process of combining a group of related variables and functions into a unit which is called an object:



    4 Pillars of Object-Oriented Programming

    Encapsulation

    IGrouping related variables and functions that operate on these variables into objects. This helps to reduce the number of parameters required by a function, because the variables can be accessed directly from within the object.

    Abstraction

    IThis means complex processes are hidden from the main function. Such as when you use imported libraries for you own code, those libraries are also using abstraction. This process helps to make the interface of the abstracted objects simpler and also this reduces the impact of change as it doesn’t leak outside to the main method.

    Inheritance

    This is the mechanism which prevents code from being redundant meaning repetition. This means properties and functions from the parent class can be inherited to the child class and this prevents the issue of repeating code

    Polymorphism

    This can be defined another way (Many forms), meaning it eliminates long conditional statements in your code as objects can be processed differently depending on their data type of class and also to redefine methods for derived classes

    I show you how you can secure on WhatsApp

    28 April 2020 / By Erika Huszar

    Be secure is always important. That’s why I decided to share this new information with you and I show you how to stay safe.


    The world's most popular messaging service is truly reliable encryption. However, a flaw allows hackers to gain full access to anyone’s account.


    It is well known how reliable end-to-end encryption of WhatsApp is, which poses a major challenge to malicious players in the digital space. if they want to access communications data. However, there is a bug in the messenger that allows for so-called watch attacks.But what it is all about? When a user logs into WhatsApp from a new device, they will need a unique code that will be sent to them in a text message. This code will allow you to recover your account settings and history from a backup on your new device.

    If, on the other hand, hackers manage to get their victim’s phone number, they can download WhatsApp to a phone and enter that certain code (which they can get from the victim’s phone if they’re spying away from them) and already have the account and all their history. The specialist who revealed the possibility of an attack also tried all this with his colleague’s phone, and everything happened as described.

    Better to safe than sorry. ~ Samuel Lover

    Let me show you how can you stay safe?


    The very first thing we can do is turn off previews in SMS. This may seem obvious, but many people want to be able to view messages even faster because people find it difficult to give up their convenience. When people use two-step authentication (also known as two-factor authentication) without an authentication app, they tend to receive codes sent via SMS, but if they can be viewed on a closed screen, it’s a bit pointless for users who have left their phone unattended.

    Therefore, secondly, never leave your phone or any device unattended. I have witnessed a lot, when people fall asleep on the train countless times while leaving their phones left on the table.


    In addition, there are a lot of malicious people in companies, so even if you trust your colleagues, there is always a chance that someone else can try this attack vector, so it’s the best thing you can do ,to never leave your device alone.


    There’s another better way to protect your account, which is better done now. WhatsApp created its own two-step authentication a few years ago for an application that can be easily tracked and prevent attack. Here's how to do it, so please open the app and set it up!


    You can set up two-step verification in WhatsApp


    First open the application .Once you have opened the application, go to Settings / Account / 2-Step Verification and click Enable. Then enter a six-digit code that you should not forget.



    Then enter your email address for additional security. Eventually, you’ll see confirmation of 2-step verification on your phone, making it much harder for someone to hijack your account or messages to another device.



    When you open WhatsApp, you will randomly request a PIN. Not every time you open the app, it shouldn’t be an inconvenience.If you pay attention and take a good safety approach, you can easily avoid it before it causes any serious damage!

    5 Tips In UI/UX Web Design

    These User Experience are waiting for us in the coming years

    28 April 2020 / By Erika Huszar

    IT is changing in a completely unpredictable way around the world, but it is certain that as more and more products are online the quality will become more important in the competition. So, anyone who wants to achieve something online, especially start-ups, has to deal with the user experience.


    Photo by pixabay

    The so-called user experience research (UX research) is a dynamically evolving industry that was created to make technology as smart, designable, enjoyable, and accessible as possible for ordinary people. To this end, future users are involved in each stage of the design, so that based on their feedback during the tests, they can finally distribute the various applications and programs widely, tailored to their needs.

    # 1 Voice UI:

    Do you know Amazon Alexa, Apple Siri and others? They are the digital assistants you can control with your voice.

    These technologies were available to 35% of U.S. households in 2019, but that rate is estimated to rise to 75% by 2025.~ https://www.statista.com/


    Advertisers of distribution companies say these gadgets are useful as they make accessing digital content much easier and simpler for a wider variety of people. For example, an elderly person does not have to constantly tap, but can use the technology with simple verbal instructions after making the basic settings, in which you can get help.These machines are so smart that if you communicate with them in a whisper, for example, due the kids sleeping in the next room, they will also lower their voices when they respond.



    # 2 Augmented Reality Technology (AR):


    You are familiar with the IKEA application that allows you to view it in your own apartment.What would the different furniture that they sell look like? Augmented reality differs from virtual reality in that we do not get into another space while completely excluding the real world, but remain in the same space, and non-real elements are projected into this real space. For example, when we are watching something through our mobile phone and our own camera is projecting images and information using an application. Let’s say we walk down the street while turning our camera towards different buildings and then the display shows the opening hours of that particular store or other information. Similar applications are currently being developed by various airlines. Apps help you find your way around the airport, find your seat, or turn your tablet or phone toward the window to find out where you are.


    Photo by Gleb Kuznetsov

    # 3 Virtual Reality Technology (VR)

    The gaming industry has been exploiting the potential of virtual reality for some time, and this technology is also becoming more convenient, smarter and cheaper. It is almost difficult to list how many places VR could be used. For example, imagine not only watching a movie, but completely getting into the story!In addition to entertainment, there is a huge learning potential in these developments. They can be used to practice various surgical procedures, and soldiers can also be spiritually prepared for a possible deployment. VR can also help us get rid of our phobias, relieve our physical pain, or just look at how well we can say no to a glass of drink after recovering from our addiction problems.

    Patrick Bordnick, dean of Tulane University’s School of Social Workers, examines in lifelike virtual spaces how well his clients are able to say no when they are offered a cigarette. In his first study, he found that smoking in cyberspace is very similar to seduction than in real space. In her later research, she found that VRE-based skills development also enabled individuals to resist smoking in high-risk situations, such as when lounging with colleagues in front of the office, waiting for a bus, or drinking alcohol. Participants were more likely not to smoke even six months after treatment. Other studies have found that VRE therapies are also effective in coming down from cocaine and gambling.


    Photo by pixabay

    # 4 Continuous design

    If you had a digital assistant, you could connect it to other devices in your smart home: with voice commands, you could control the thermostat, air conditioner, lights, sockets, cameras based on the instructions you gave. Users these days are increasingly expecting everything to be wired together. Continuous design helps with data synchronization, which means that individual elements of the system can be upgraded and replaced individually, without having to redesign the whole thing. It also follows that when we get the latest version from an app or a widget, we can easily sync again with the other technologies we use again. For example, when switching from one medium to another, we can usually do so without interruption, without having to always log in and out of different systems. Of course, all this raises serious privacy concerns.


    Photo by pixabay

    # 5 User experience metrics

    UX research is now developed, with the help of various programs, they can associate a score with products that characterizes them based on the quality of the user experience they provide. This includes behavioural data (how popular the products are) as well as attitude data (how easy they are to use; how reliable they work and how attractive their workmanship is). This is also useful because investors also receive feedback, so you can also instruct consumers on what programs, applications and gadgets are worth buying.

    Do we really need all this?

    Among the technologies listed above are those that seem to be a higher priority (such as innovations related to training, healing), and there are some that are more for our entertainment and comfort. With regard to the latter technologies, however, the question arises as to whether it is really so important to pull down the shutter with an application or to turn on the lights. Should everything be even faster when our lives are already completely up and running?

    At i = 2 the following next two values 4 and 5 are already in order and therefore 4 < 5 is not swapped: [2, 4, 5, 3, 9]


    #2 nd for loop of 4th iteration passing through the list:


    At i = 3 the next two values, 5 and 3, are not in order and therefore the if statement 3 < 5 is true and executed, so the algorithm swaps them: [2, 4, 3, 5, 9] restarting the for loop again.


    #3 rdfor loop of 2nd iteration passing through the list:


    At i = 2 the two numbers 4 and 3, are not in order and therefore the if statement 3 < 4 is true and executed, so the algorithm swaps them: [2, 3, 4, 5, 9]

    #3rd for loop of 4th iteration passing through the list:


    At i = 4 getting the numbers 5 and 9 and executing the if statement based on the algorithm, 9 < 5 is not true and therefore these numbers are in order and not swapped: [2, 4, 3, 5, 9]

    Now the list is arranged, however the bubble sort algorithm doesn't understand this. Or maybe, it needs to finish a whole go through the list without swapping any value to realize the list is arranged.

    #last iteration passing through the list:


    • [2, 4, 3, 5, 9] => [2, 4, 3, 5, 9]
    • [2, 4, 3, 5, 9] => [2, 4, 3, 5, 9]
    • [2, 4, 3, 5, 9] => [2, 4, 3, 5, 9]
    • [2, 4, 3, 5, 9] => [2, 4, 3, 5, 9]
